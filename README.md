# Inventory Management System (IMS)

A web-based inventory management system built with **React** and **Firebase**. This application allows businesses to track product inventory, manage stock levels, and get dashboard insights into their operations. This project was developed as a **final group project** for the **Web and Mobile Application Development** course under the **Bachelor of Science in Information Technology (BSIT)** program at **Bulacan State University**, demonstrating the application of UI/UX design, database management, and full-stack programming skills.

---

## About This Project

This serverless, web-based application solves common inventory challenges with a clean, responsive interface for managing products, tracking stock, and visualizing inventory health in real-time. It prioritizes **secure authentication**, a **reliable database backend**, and a **reactive, component-based frontend**.

---

## Core Features

- **Authentication**: Secure registration/login (Email/Password & Social), email verification, and protected routes.  
- **Onboarding**: Multi-step flow capturing personal profile info.  
- **Inventory Management**: Full CRUD for products with fields like Product Type, Quantity, Price, and Minimum Level.  
- **Dashboard**: Displays key metrics:
  - Total Inventory Value  
  - Total Units in Stock  
  - Low Stock Item Count  
- **Visualizations**: Responsive charts using **Recharts**.  
- **Responsive Design**: Built with **Bootstrap** and custom CSS.

---

## Key Components

- **AuthenticationLayout**: Route protection and redirects.  
- **VerificationForm**: Handles email verification.  
- **PersonalInfoForm**: Captures user details post-registration.  
- **InventoryDashboard**: Shows KPIs and stock charts.  
- **Sidebar**: Collapsible navigation.

---
